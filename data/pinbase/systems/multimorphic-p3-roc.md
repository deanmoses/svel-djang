[[manufacturer:multimorphic]]'s next-generation pinball control platform, designed specifically for their P3 modular pinball platform rather than retrofitting existing machines. It uses an FPGA-based architecture paired with a host computer running Linux and features an integrated LCD playfield display capability. The P3 platform debuted with [[title:lexy-lightspeed]]: Escape from Earth and supports swappable game modules.
